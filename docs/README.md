# はじめに

この資料は静岡プログラミングキャンプ サーバサイドプログラミングコースの資料になります。

## 課題の進め方について

進め方大きく分けて、2つのステップに分かれています。

1. 既存のコードを改修してより良いコードを目指す
1. 与えられた課題について自分なりに設計し、実装する。

この２つのステップで学ぶことによってより良いコードを書くためスキルアップに繋がります。
